# UZ-Olymp Open Archive

A centralized, structured archive for Uzbekistan Olympiad of Informatics contests and related competitive programming events.

## Purpose

This repository preserves official materials and model solutions for:

* IOI-TST selections
* National and regional Olympiads
* KHIMIO and special contests
* Future seasonal competitions

## Naming Rules

* Folder: problem name 
* Source: taskname.cpp
* Statement: statement.pdf

## Contribution Guidelines

* Follow existing structure
* Keep formatting consistent
* Do not modify original statements

## Maintainer

Uzbekistan Competitive Programming Community

Mission: Preserve official materials and provide clean, reliable references for current and future generations.
